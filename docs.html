Keyboard Shortcuts

  • j/k - Navigate up/down
  • f - Show link hints
  • r - Toggle recording
  • p - Play recorded sequence
  • c - Clear recorded sequence

Navigation Modes

  • Cursor Mode - Visual cursor for element selection
  • Highlight Mode - Highlight current element

  • Record keyboard sequences
